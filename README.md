# Project Ad Marketspace

### Made by:
#### * Andre Novado
#### * Bagas Kurniajati
#### * Muhammad Sholeh

The purpose of this project is to create an ad marketplace where publisher and advertiser can do their business online.
This project is created using:
- Front End:
  - React with Redux
  - HTML, CSS
  - MDBootstrap
- Back End:
  - Python Flask
- Database:
  - MySQL
- Deployment:
  - AWS Server
  - Nginx Virtual Machine
  - Travis CICD
  - Kubernetes
